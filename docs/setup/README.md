# Setup Documentation

This folder contains setup and installation guides for ClipSync.

## Contents

- **[QUICK-START.md](./QUICK-START.md)** - Quick start guide to get up and running
- **[LOCAL-SETUP.md](./LOCAL-SETUP.md)** - Detailed local development setup
- **[SETUP.md](./SETUP.md)** - General setup instructions
- **[SETUP-COMPLETE.md](./SETUP-COMPLETE.md)** - Setup completion checklist
- **[ELECTRON-START.md](./ELECTRON-START.md)** - Electron desktop app setup

## Getting Started

1. Start with [QUICK-START.md](./QUICK-START.md) for the fastest setup
2. For detailed setup, see [LOCAL-SETUP.md](./LOCAL-SETUP.md)
3. Verify setup with [SETUP-COMPLETE.md](./SETUP-COMPLETE.md)

## Prerequisites

- Node.js 18+ LTS
- Docker Desktop (for local database)
- Git
- Code editor (VS Code recommended)
