ğŸ”´ Launcher Window Closes Immediately:

If the window flashes and disappears:
1. Use launcher.bat instead of START.bat (better error handling)
2. Check launcher-error.log file (errors are logged there)
3. Run from command prompt:
   - Open CMD
   - Navigate to ClipSync folder
   - Type: START.bat
   - This will show any errors before closing

ğŸ”´ "Docker is not running":

- The launcher will continue even if Docker isn't running

ğŸ”´ "Node.js not found":
- Install Node.js from https://nodejs.org/
- Restart your computer after installation
- Verify with: node --version in command prompt

ğŸ”´ Port already in use:

- Check if ClipSync is already running

ğŸ”´ Services won't start:
