# Keyboard Shortcuts Reference

## Global Shortcuts

### Application
- `Ctrl+Shift+V` (Windows/Linux) / `Cmd+Shift+V` (Mac) - Open ClipSync
- `Ctrl+K` - Quick search
- `Ctrl+N` - New clip
- `Escape` - Close dialog/modal

### Navigation
- `Ctrl+ArrowDown` - Next clip
- `Ctrl+ArrowUp` - Previous clip
- `Ctrl+P` - Pin/unpin clip
- `Delete` - Delete clip

### Clipboard
- `Ctrl+C` - Copy
- `Ctrl+V` - Paste
- `Ctrl+Shift+C` - Copy to ClipSync

## Browser Extension

- `Ctrl+Shift+V` - Open extension popup
- `Ctrl+Shift+H` - Quick paste from history

## VS Code Extension

- `Ctrl+Shift+P` → "ClipSync: Open Panel" - Open ClipSync panel
- `Ctrl+Alt+V` - Quick paste from history

## CLI

```bash
clipsync copy "text"       # Copy to clipboard
clipsync paste             # Paste from history
clipsync search "query"    # Search clips
clipsync list              # List recent clips
clipsync interactive       # Interactive mode
```

## Customization

All keyboard shortcuts can be customized in Settings → Keyboard Shortcuts.

---

**Note**: Shortcuts may vary by platform. Mac users should use `Cmd` instead of `Ctrl`.
