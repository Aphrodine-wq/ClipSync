# ClipSync User Guide

## Welcome to ClipSync!

ClipSync is your professional clipboard manager that syncs across all your devices.

## Getting Started

### 1. Sign In
- Click "Sign in with Google"
- Authorize ClipSync
- Start using immediately!

### 2. Install Extensions
For seamless clipboard capture:
- **Chrome**: Install from Chrome Web Store
- **Firefox**: Install from Firefox Add-ons
- **Desktop**: Download from website
- **Mobile**: Download from App Store/Play Store

### 3. First Clip
Just copy something! ClipSync automatically captures and syncs your clipboard.

## Core Features

### Searching Clips
- **Quick Search**: Press `Ctrl+K` (Windows/Linux) or `Cmd+K` (Mac)
- **Filter**: Filter by type, tags, date
- **Semantic Search**: Use natural language queries
