# ClipSync Brand Guidelines

## Brand Identity

### Logo
**Primary Logo**: ClipSync wordmark with a sync/clipboard icon
**Icon**: A stylized clipboard with synchronization arrows
**Color Variations**:
- Primary: Full color (blue gradient)
- Dark mode: White on dark background
- Monochrome: Single color for print

### Color Palette

**Primary Colors**
- **Primary Blue**: `#3B82F6` (Modern, tech-focused)
- **Secondary Blue**: `#1E40AF` (Deeper blue for emphasis)
- **Accent Purple**: `#8B5CF6` (For gradients and highlights)

**Secondary Colors**
- **Success Green**: `#10B981` (Sync status, positive actions)
- **Warning Amber**: `#F59E0B` (Alerts, warnings)
- **Error Red**: `#EF4444` (Errors, delete actions)

**Neutral Colors**
- **White**: `#FFFFFF`
- **Light Gray**: `#F3F4F6`
- **Medium Gray**: `#9CA3AF`
- **Dark Gray**: `#374151`
- **Black**: `#111827`

### Typography

**Headlines**
- Font: Inter or System UI
- Weight: 600-700 (SemiBold to Bold)
- Size: 24px - 72px

**Body Text**
- Font: Inter or System UI
- Weight: 400-500 (Regular to Medium)
- Size: 14px - 18px

**Code/Monospace**
- Font: JetBrains Mono, Fira Code, or monospace
- Use for technical content, code snippets, keyboard shortcuts

### Voice & Tone

**Brand Voice**: Professional, approachable, developer-focused

**Tone Guidelines**:
- **Direct**: Get to the point quickly
- **Expert**: Show we understand developers' needs
- **Empowering**: Help users be more productive
- **Modern**: Use contemporary language, avoid jargon

**Examples**:
✅ "Your clipboard, everywhere you need it."
✅ "Boost your productivity with smart sync."
❌ "ClipSync represents a paradigm shift in clipboard management."

### Copywriting Guidelines

**Headlines**:
- Keep under 10 words when possible
- Focus on benefits, not features
- Use active verbs

**Product Descriptions**:
- Start with user benefit
- Follow with how it works
- Include specific use cases

**Technical Content**:
- Use code examples
- Include keyboard shortcuts
- Provide step-by-step instructions

## Visual Elements

### Icon Style
- Use Lucide icons or similar clean, modern icon set
- Consistent stroke width (2px)
- Rounded corners for friendlier appearance

### Photography Style
- Clean, minimal screenshots
- Professional workspace environments
- Diverse representation of developers
- Modern technology setup

### Illustration Style
- Clean, flat design
- Abstract tech concepts
- Subtle gradients
- Professional color palette

## Do's and Don'ts

### Do's
✅ Use proper spacing and whitespace
✅ Maintain visual hierarchy
✅ Keep text readable (contrast ratio 4.5:1)
✅ Use the full color palette
✅ Include code examples for developer features

### Don'ts
❌ Stretch or distort the logo
❌ Use brand colors excessively
❌ Create complex, cluttered designs
❌ Use generic stock photos
❌ Make technical claims without evidence
